# Changelog

All notable changes to this project will be documented in this file.

## [2.0.0-beta] - 2026-05-15

### Major Changes
- **Rebranding**: Officially renamed the application from **SDAHymns** to **Hymnia**.
- **Visual Identity**: New high-resolution application icon.

### New Features
- **Auto-Updates**: Integrated **Velopack** for seamless, background application updates.
- **Cross-Platform Support**: 
    - **First Linux Release**: Native support for Linux with AppImage and portable tarball options.
    - **ARM64 Support**: Native ARM64 builds for both Windows and Linux (perfect for Surface Pro, Raspberry Pi, and other ARM devices).
- **Architecture Detection**: The installer now warns if you are running the wrong version for your machine's architecture to ensure optimal performance.

### Technical Updates
- Migrated to .NET 10.
- Updated all dependencies to the latest versions.
- Refactored project structure to match the new Hymnia branding.
